# nanonama
A tiny message machine 
 -. .- -. --- -. .- -- .-

Read / write via MQTT.
Write via one morse button
Allows very tiny board. i.e. ESP32c3 with 0.42" OLED.
Config via onboard HTTP-server.

# Build
Needs to install ESP-IDF <br>
then: <br>
<br>
 source <path-to-esp-idf>/export.sh<br>
 idf.py build<br>
 idf.py flash
